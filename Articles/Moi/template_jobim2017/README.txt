1 English version
=================

  The papers should be submitted using the provided templates.  The
  templates differ depending on the submission type and can be found in
  the subdirectories.
  `Poster-Demo': For both /Poster/ and /Demo/ submissions
  `Highlight': For the /Highlight/ submissions
  `Proceedings': For the /Proceedings/ submissions


  The docx and odt templates provide existing styles (whose names start
  with `Jobim') which must be used. Each template shows how each style
  is used.

  The LaTeX template relies on the `jobim2017' class (provided with the
  templates). The paper should be written following the comments within
  the `.tex' file.

  The LaTeX template has been successfully tested with the online LaTeX
  editors ShareLaTeX and Overleaf.

  In case you encounter the error message `Metric (TFM) file not
  found.', you should install the `Times' font:
  Linux: 
         - `apt-get install texlive-fonts-recommended' (Debian/Ubuntu)
         - `dnf install texlive-collection-fontsrecommended.noarch' (Fedora)
  Mac OS X: `tlmgr install collection-fontsrecommended'


2 Version française
===================

  Les articles doivent être soumis en utilisant les modèles fournis. Les
  modèles varient en fonction du type de soumission. Ces modèles se
  trouvent dans les sous-répertoires :
  `Poster-Demo': Pour les soumissions de type /Poster/ et /Demo/
  `Highlight': pour les soumissions de type /Highlight/
  `Proceedings': pour les soumissions de type /Proceedings/

  Les modèles docx et odt fournissent les styles à utiliser (leurs noms
  commencent par `Jobim'). Chaque modèle montre comment ces styles sont
  utilisés.

  Le modèle LaTeX utilise la classe `jobim2017' (fournie avec les
  modèles). Les articles doivent être écrits en suivant les consignes
  donnés en commentaires dans le fichier `.tex'.

  Au cas où vous rencontrez un message d'erreur `Metric (TFM) file not
  found', vous devriez installer la police `Times' :
  Linux: 
         - `apt-get install texlive-fonts-recommended' (Debian/Ubuntu)
         - `dnf install texlive-collection-fontsrecommended.noarch' (Fedora)
  Mac OS X: `tlmgr install collection-fontsrecommended'
